## Overview

InfraGuard integrates with Grafana to provide rich visualizations of anomaly detection, forecasts, and alert history.

## Setup

<Steps>
  <Step title="Create API Key">
    1. In Grafana, go to Configuration → API Keys
    2. Click "Add API key"
    3. Name: "InfraGuard", Role: "Editor"
    4. Copy the API key
  </Step>
  
  <Step title="Configure InfraGuard">
    ```yaml
    integrations:
      grafana:
        enabled: true
        url: "http://grafana:3000"
        api_key: "${GRAFANA_API_KEY}"
        dashboard_uid: "infraguard-overview"
    ```
  </Step>
  
  <Step title="Import Dashboards">
    ```bash
    infraguard grafana import-dashboards
    ```
  </Step>
</Steps>

## Custom Panels

Create custom panels using InfraGuard's Prometheus metrics:

```promql
# Anomaly score over time
infraguard_anomaly_score{metric="cpu_usage"}

# Forecast accuracy
infraguard_forecast_mape{metric="memory_usage"}

# Alert counts by severity
sum by (severity) (infraguard_alerts_total)
```

## Annotations

InfraGuard automatically adds annotations to Grafana for:

- Anomaly detections
- Alert triggers
- Model retraining events
- Configuration changes
